// Como instalar

  1. Gmail: Configurações (⚙) → Ver todas as configurações → Assinatura → Nova → clique em "Copiar assinatura" abaixo e cole.
  2. Outlook Desktop: Arquivo → Opções → Email → Assinaturas → Nova → cole no editor HTML.
  3. Apple Mail: Mail → Preferências → Assinaturas → clique em + → edite nome → arraste o HTML para a área de assinatura.
  4. Substitua os campos [ENTRE COLCHETES] pelos dados reais antes de instalar.
  5. A assinatura usa fontes de fallback (Impact, Courier New) quando Google Fonts não carrega — funciona offline.

Layout table-based: compatível com Gmail · Outlook 2016+ · Apple Mail · Thunderbird · Superhuman. Sem imagens externas — zero bloqueio por firewall corporativo.
